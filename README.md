**Foodie's Hub – Online Food Ordering Website**

Foodie's Hub is a responsive web app where users can browse food, search items, add them to a cart, and place orders.

**Features**

-Responsive design for desktop & mobile

-Menu with Main Course, Appetizers, Beverages

-Search food items by name

-Add to cart and view selected items

-Smooth scroll navigation

-Order and contact forms

**Tech Stack**

HTML, CSS, JavaScript

**Usage**

-Clone the repo:

git clone https://github.com/yourusername/foodies-hub.git

-Open Foodie's Hub.html in your browser

-Browse menu, search, add to cart, and place orders
